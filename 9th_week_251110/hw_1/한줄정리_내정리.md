FastAPI<br>
: 파이썬으로 만들어진 경량화된 마이크로 웹 프레임워크<br>
pyproject.toml	<br>
: 파이썬 프로젝트의 버전 호환성 및 타인 공유를 위한 프로젝트 사양서<br>
Gunicorn<br>
: 작성된 코드를 웹상에 배포할 수 있도록 도와주는 WSGI HTTP 서버 엔진<br>
HTTP Message	<br>
: 웹에서 클라이언트와 서버간에 소통할 수 있는 메시지 형식<br>
HTTP Request Method	<br>
: 클라이언트와 서버간에 메시지를 주고 받을때의 타입 형식의 정의<br>
HTTP Status Code	<br>
: 서버 응답시 서버에서 클라이언트로 보내주는 상태 코드(200 정상, 404 파일 찾을 수 없음 등이 있다)<br>
HTTP URL	<br>
: 특정 웹 서버의 주소를 문자열로 표현해주는 방법<br>
Query string 	<br>
: 클라이언트에서 서버로 요청을 보낼때 부수적으로 보내는 데이터의 전송 규격<br>
Path Variable	<br>
: 클라이언트에서 서버로 요청을 보낼때 특정한 경로를 서브 주소로 보내기 위해 사용하는 변수<br>
JSON	<br>
: 데이터를 구조화하기 위해 사용되는 규격(Javascript Object notation)<br>
REST API	<br>
: 클라이언트와 웹이 서로 요청/응답을 받을 때 CRUD를 고려해서 설계하는 API 형태<br>
디자인패턴	<br>
: 하나의 어플리케이션의 구성 및 연결, 외부와의 연결을 위해 개발할 때 사용되는 패턴의 총칭<br>
Database	<br>
: 자료들을 일정한 정보 형태로 변환하여 저장해 놓는 공간<br>
SQL	<br>
: DB를 구조화해서 그 안의 자료를 조작하기 위해 사용하는 언어<br>
데이터 정의어	<br>
: 테이블을 생성하거나 수정하는 작업에서 사용하는 명령어들의 집합<br>
데이터 조작어	<br>
: 데이터를 추가, 수정, 삭제 할 때 사용하는 명령어들의 집합<br>
데이터 제어어	<br>
: 테이블의 권한을 부여, 해제하기 위해 사용하는 명령어들의 집합<br>
JOIN	<br>
: 2개 이상의 테이블을 결합할 때 사용하는 SQL문의 명령어<br>
정규화	<br>
: 데이터의 범위를 재구성해서 각 데이터간의 비교가 용이하도록 만들어주는 과정<br>
암호화 	<br>
: 데이터의 보안을 위해 별도의 해시값을 기반으로 재구성하는 과정 <br>
bcrypt	<br>
: 데이터를 base64 기반의 암호문으로 변경할 때 사용하는 라이브러리<br>
stateful	<br>
: 웹 페이지 내부에서 외부와의 연결을 통해 구성 컴포넌트들이 동적으로 변경되는 페이지를 의미<br>
stateless	<br>
: 웹 페이지 내부에서 외부와의 연결을 통해 구성 컴포넌트들이 동적으로 변경되지 않고 정적인 상태에서 자체가 변경되는 페이지를 의미<br>
connectionless	<br>
: ??<br>
인증·인가	<br>
: 사전에 허가 받은 사용자에게 권한을 부여하는 과정 및 이를 허가하는 과정을 의미<br>
쿠키	<br>
: 특정 페이지에서 자주 사용되는 데이터를 웹 브라우저 상의 내부 스토리지에 저장해서 서버의 호출없이 사용할 수 있도록 도와주는 기능<br>
세션	<br>
: 사용자가 웹 프로그램에서 접속해서 활동을 하는 일련의 과정을 의미<br>
프레임워크	<br>
: 개발시 자주 사용되는 함수나 데이터를 한 곳에 모아둔 집합의 형태???<br>
라이브러리	<br>
: 개발시 자주 사용되는 함수나 데이터를 한 곳에 모아둔 집합의 형태???<br>
API	<br>
: 서로 다른 2개의 어플리케이션이 통신하기 위해 사용하는 공통의 규약<br>
페이징	<br>
: 하나의 페이지에서 다른 페이지로 넘어가는 동작을 의미<br>
HTTPS	<br>
: 기존 HTTP에 보안이 강화된 접속 프로토콜<br>
log	<br>
: 어플리케이션의 구동시 발생될 수 있는 모든 동작을 기록화해둔 것<br>
HTML	<br>
: 웹 어플리케이션의 내용을 작성할 때 사용하는 마크업 언어<br>
CSS	<br>
: HTML 의 요소들에 디자인적인 부분을 반영할 수 있는 스크립트 언어<br>

Grid	<br>
: Flex와 유사하나 미리 설정된 구역 형태 안에서 컴포넌트를 배치하는 디스플레이 방식<br>

Search Engin Optimization	<br>
